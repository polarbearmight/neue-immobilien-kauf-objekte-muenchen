# Fitness Tracking (GymBro)

## Ziel
**Cut** (fokussiert auf Fettverlust mit maximalem Muskelerhalt).

Aktueller Zielrahmen:
- Start: **111.8 kg** bei **200 cm** (2026-03-03)
- Ziel: **105.0 kg bis 2026-05-15**
- Erforderliche Rate: ca. **0.65 kg/Woche** (~0.58% KG/Woche)


## Erfasste Daten (pro Tag)
- Gewicht (kg)
- FDDB: Kalorien, Protein, Kohlenhydrate, Fett
- Training: Übungen, Sätze, Gewichte, Dauer, RPE (optional)
- Optional: Schritte, Schlaf, Befinden

## Dateiübersicht
- `daily-log.jsonl` → fortlaufender Tages-Log (ein JSON-Objekt pro Eintrag)
- `checkins.md` → kompakte, menschenlesbare Historie

## Hinweis
GymBro gibt nach jedem Log-Eintrag kurzes Feedback und erstellt alle 14 Tage eine Analyse.
